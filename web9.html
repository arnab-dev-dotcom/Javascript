ES6

ES6 (ECMAScript 2015) is a significant update to JavaScript that introduced many new features and improvements, making the language more powerful and easier to work with. Some of the most notable features of ES6 include:
  1. Destructuring Assignment

      Allows unpacking values from arrays or objects into distinct variables.

  2. Spread Operators

    Spread ... allows you to unpack elements from an array or object.

  3. Reset Operators

    Reset ... allows get element and pack them into an array or object.

  4. Nullish Operator

    The nullish coalescing operator (??) in JavaScript is used to provide a default value when dealing with null or undefined. It returns the right-hand operand only if the left-hand operand is null or undefined; otherwise, it returns the left-hand operand.

  5. Optional Chaining (?.)

    In JavaScript The optional chaining (?.) operator allows you to safely access deeply nested object properties without throwing an error if a property is null or undefined. If the property exists, it returns its value; otherwise, it returns undefined.

  6. Set

    A Set in JavaScript is a built-in object that allows you to store unique values of any type (primitive or object). It automatically removes duplicates.

  7. ✔ Use for...in when looping over objects. ✔ Use for...of when looping over arrays, strings, Maps, or Sets.
