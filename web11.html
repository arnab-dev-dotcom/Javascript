Array & Array Method

Array is a linear data structure, In JS array hold multiple types of data in contigious memory location

Define Array:

In JS array define as []

push():

push() method used to Insertion element to the array

pop():

pop() method used to Deletion element from end of the array

length():

length() method is used to define the length of the array

unshift():

unshift() method is used to insert the element to the begining of the array

shift():

shift() method is used to delete the element to the begining of the array

indexOf():

indexOf() method is used to find the index of an element

isArray():

isArray() method value type is array or not

join():

The join() method in JavaScript is used to combine all elements of an array into a single string, with a specified separator between them.

slice():

In JavaScript, slice() is a method used to extract a section of an array or string and return it as a new array or string, without modifying the original.

splice():

splice() modifies the original array. It can be used to add, remove, or replace elements in-place.

concat():

In JavaScript, concat() is used to merge arrays or strings without modifying the originals. It returns a new array or string.
