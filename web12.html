MODULES

JavaScript modules are a way to break your code into reusable pieces

Why Use Modules?

Code Reusability: Split code into files that can be reused across projects.
Encapsulation: Each module has its own scope.
Maintainability: Smaller files are easier to maintain.

Note: In HTML, make sure to add type="module" to the script tag:

In JavaScript, default is a keyword used when exporting or importing a default value from a module. It allows you to export one primary thing from a module, and then import it without using curly braces. You can change the name also
